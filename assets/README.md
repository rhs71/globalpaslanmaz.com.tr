# Assets Folder

This folder contains images, logos, and other static assets for the Global Paslanmaz website.

## Required Files

Please add the following files to this folder before deploying:

### 1. Logo
- **Filename**: `logo-global-paslanmaz.png`
- **Recommended size**: 200x50px (or similar aspect ratio)
- **Format**: PNG with transparent background
- **Usage**: Header, footer, and branding

### 2. Favicon
- **Filename**: `favicon.png`
- **Size**: 32x32px or 64x64px
- **Format**: PNG or ICO
- **Usage**: Browser tab icon

### 3. Product Images (Optional)
You can add product images here and reference them in the HTML files:
- `product-tezgah-1.jpg`
- `product-davlumbaz-1.jpg`
- etc.

## Image Optimization Tips

Before adding images:
1. **Compress** images to reduce file size (use TinyPNG, ImageOptim, or similar tools)
2. Use **appropriate formats**:
   - PNG for logos and graphics with transparency
   - JPG for photographs
   - WebP for modern browsers (optional, with fallbacks)
3. **Resize** images to the actual display size needed
4. Use **descriptive filenames** (e.g., `calisma-tezgahi-paslanmaz.jpg`)
